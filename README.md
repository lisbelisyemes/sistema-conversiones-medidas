# Proyecto 1: Sistema de Conversiones y Equivalencias de Medidas y Pesos de los Ingredientes de la Comida

Este proyecto fue desarrollado como parte de la evaluación de la asignatura **Técnicas de Programación 3** de la Universidad Nacional Experimental de Guayana (UNEG).

El objetivo principal es aplicar los conceptos de Programación Orientada a Objetos (POO) como herencia, polimorfismo y encapsulamiento en una aplicación funcional con interfaz gráfica.

## Descripción del Programa

El software permite realizar conversiones entre diferentes sistemas de unidades (Sistema Internacional y Sistema Inglés) para tres magnitudes físicas utilizadas en gastronomía:

* **Masa/Peso:** Convierte entre Kilogramos, Libras, Gramos y Onzas.
* **Volumen:** Incluye Litros, Mililitros, Tazas (Medida estándar americana y métrica), Cucharadas y Cucharitas.
* **Temperatura:** Convierte entre grados Celsius y Fahrenheit.

El programa valida que los datos ingresados sean numéricos y maneja errores para evitar cierres inesperados.

## Herramientas Utilizadas

* **Lenguaje:** Java.
* **Entorno de Desarrollo (IDE):** NetBeans 8.0.2 (o superior).
* **Interfaz:** Java Swing.
* **Diseño:** Librería externa "FlatLaf" para mejorar la apariencia visual.

## Instrucciones de Instalación y Ejecución

Para ejecutar este proyecto en su computadora, siga estos pasos:

1.  **Descargar:** Haga clic en el botón verde "Code" y seleccione "Download ZIP", o clone el repositorio.
2.  **Descomprimir:** Extraiga los archivos en una carpeta de su preferencia.
3.  **Abrir en NetBeans:**
    * Inicie NetBeans.
    * Vaya a `File` -> `Open Project`.
    * Busque la carpeta del proyecto y selecciónela.
4.  **Verificar Librerías:**
    * El proyecto incluye una carpeta llamada `lib` con el archivo `flatlaf.jar`.
    * NetBeans debería reconocerla automáticamente. Si marca error, haga clic derecho en el proyecto -> `Properties` -> `Libraries` y agregue el archivo JAR que está dentro de la carpeta `lib`.
5.  **Correr el programa:**
    * Ubique el archivo `Main.java` dentro del paquete `app.conversionmedidas`.
    * Haga clic derecho y seleccione **Run File**.
